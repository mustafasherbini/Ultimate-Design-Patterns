# Design Patterns

I'm studying Design Patterns using the course [Ultimate Design Patterns](https://www.udemy.com/course/ultimate-design-patterns/). Here I'm solving the exercise of each pattern, implementing solutions in both C++ and C#, with LLM assistance for refactoring and code review. I also document the key differences between C++ and C# implementations to deepen my understanding of C++. Each pattern includes a UML diagram to visualize the design.

The main reason for documenting this is to help me recap the design patterns again in the future, and to recap the differences between C++ and C#.
